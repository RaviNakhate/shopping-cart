## shopping-cart
 - It is shopping cart functionality of e-commerce site.
 
## Installation
```bash
git clone https://github.com/RaviNakhate/shopping-cart.git
```

## Install Package
```bash
npm install react-router-dom
```
```bash
yarn add react-router-dom
```
